# EverFresh Dairy E-Commerce Platform

EverFresh is a modern dairy and frozen products management platform designed for showcasing and managing premium dairy products, frozen foods, product variants, categories, and customer inquiries.

---

## 🚀 Technology Stack

### Frontend
- React.js
- Tailwind CSS
- JavaScript (ES6+)
- Axios

### Backend
- Laravel 12/13
- PHP 8.4
- REST API Architecture

### Database
- MySQL

### Server
- Apache Server

### Development Tools
- Composer
- NPM
- Git & GitHub
